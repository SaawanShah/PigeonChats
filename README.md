##  Developer

##  Overview
Pigeon Chat App is a real-time messaging application built using Kotlin and Firebase. The app enables users to communicate instantly with a smooth and responsive user experience while following modern Android development practices.

##  Features
- User Authentication (Login/Signup)
- Real-time Messaging using Firebase
- Clean and intuitive UI
- Efficient data handling
- Error handling and validation

##  Tech Stack
- Language: Kotlin
- Architecture: MVVM
- Backend: Firebase (Authentication & Realtime Database)
- UI: XML (Material Design)

##  Architecture
The project follows MVVM architecture to ensure separation of concerns, scalability, and maintainability.

##  How to Run
1. Clone the repository  
2. Open the project in Android Studio  
3. Add your `google-services.json` file (Firebase setup required)  
4. Sync Gradle and run the app  

##  Note
Firebase configuration is required to run this project properly.
